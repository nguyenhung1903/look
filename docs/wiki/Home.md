# look Wiki

Welcome to the `look` project wiki.

`look` is a keyboard-first macOS launcher focused on fast local actions with a lightweight Rust core.

## Who this wiki is for

- **Users**: learn install, usage, shortcuts, and configuration.
- **Contributors**: learn architecture, backend boundaries, and safe edit targets.

## Core concepts

- local-first indexing and ranking,
- Swift UI shell + Rust engine via FFI,
- bounded query-time scoring for stable latency,
- usage-aware ranking feedback loop.
