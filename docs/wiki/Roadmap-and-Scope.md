# Roadmap and Scope

## Current focus

- fast macOS launcher workflow
- stable local indexing and ranking
- practical built-in commands

## Near-term priorities

- deeper System Settings coverage
- safer shell execution controls
- richer benchmark reporting (p50/p95/p99)
- scoring calibration improvements

## Longer-term direction

- incremental indexing via file system events
- optional extension/plugin injection model
- Windows after macOS quality stabilizes

## Out of scope for v1

- cloud-first architecture
- semantic/vector retrieval
- full file content indexing
